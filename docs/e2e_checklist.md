# E2E Checklist

## Case 1: 小红书发布（需验证码）
- 输入：`需要发布长沙旅游景点的小红书帖子，手机号 15007473274`
- 预期：
  - Agent 输出计划事件（plan_created）
  - 自动执行到“填手机号 -> 勾选同意 -> 点击获取验证码”
  - 状态切换到 `waiting_user` 并请求验证码

## Case 2: 非阻断任务无需提问
- 输入：`帮我总结长沙旅游景点并生成一段发布文案`
- 预期：
  - Agent 直接执行搜索与文案生成，不询问无关问题
  - 最终状态 `completed`

## Case 3: 工具异常自恢复
- 制造条件：断开浏览器/设备会话
- 输入：`继续发布流程`
- 预期：
  - 日志出现失败与决策摘要
  - Agent 自动重建会话并重试关键步骤
  - 不出现无限重复起点行为
